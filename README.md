邦尼 ERP 系統是一款為中小企業打造的簡易網頁版 ERP 系統，使用 **HTML5 + Bootstrap 5 + jQuery** 建構，用於管理商品庫存資料，提供基本的資料新增、修改、刪除功能。

---

## 📦 功能介紹

### ✅ 基本功能

- 新增商品資料
- 點擊表格資料 → 自動帶入輸入欄位（修改前置）
- 清空輸入欄位
- 刪除任一筆資料（含動態新增資料）

### 🖥️ 使用技術

- HTML5
- Bootstrap 5.3.3（CDN）
- jQuery 3.7.1（CDN）

---

## 🏗️ 系統模組與說明

| 模組         | 功能簡介 |
|--------------|----------|
| 資料表格顯示 | 顯示商品庫存紀錄，含操作按鈕（刪除） |
| 表單區塊     | 使用者可輸入商品基本資料 |
| 按鈕控制     | `添加資料`：將輸入內容加入表格<br>`清空上面`：清除輸入欄位資料 |
| jQuery 功能  | 處理資料填入、刪除、表格行點擊事件等 |

---

## 🚀 使用方式

1. 下載或 clone 專案
2. 使用瀏覽器開啟 `index.html`
3. 開始操作商品管理功能

---

## 📊 ERP 系統特色介紹（網頁內容）

- 模組化架構，依企業需求靈活配置
- 支援雲端與本地部署
